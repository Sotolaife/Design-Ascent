# **App Name**: Design Ascent

## Core Features:

- Hero Section Design: Premium layout with a bold headline and clear CTA to register for the free graphics design class.
- Instructor Credibility Showcase: Highlight the instructor's experience as Creative Director at Adesina Assets and their work with top brands using minimalist icons and brand logos.
- Curriculum Preview: Outline what participants will learn in the free class, focusing on practical skills and earning potential. It touches upon the fundamentals of graphics design and client expectations.
- Transformation Section: Illustrate the opportunity to learn from an industry expert and enter a network of creatives.
- Class Details Display: Present class schedule and platform (Zoom, Google Meet), displayed in a clean schedule layout.
- Registration Form: Collect user information to reserve a spot for the free class and include an element of urgency to maximize conversions.
- Final Call-to-Action: End with a concluding section and strong CTA button, motivating the user to register.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5), a desaturated version of the primary color, to provide a clean and modern backdrop.
- Accent color: Gold (#FFD700) to highlight key elements such as CTAs and create a premium feel.
- Headline font: 'Poppins' for headlines and short descriptions to give a precise, contemporary feel.
- Body font: 'Inter' for longer descriptions, as this will give the website a modern, neutral look.
- Minimalist icons related to graphics design, brands, and skill acquisition to reinforce content.
- Clean and spacious layout with a clear visual hierarchy to guide the user's eye to key information and calls to action.
- Subtle animations on hover and scroll to add interactivity without distracting from the content.
